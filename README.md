Welcome to the official repository for the **Cloudville City Map**! This project contains all the geographic data and project files used to map out Cloudville. 

---

## 📂 Repository Structure

All mapping files are organized within the `QGIS_files` directory:

* **`*.qgz` (QGIS Project File):** Use this file if you want to make changes, updates, or additions to the map.
* **`*.gpkg` (GeoPackage File):** Contains the actual map data; ideal for visualization or integration with other GIS and mapping software.

---

## 🤝 Contributing & Editing

We welcome and encourage community contributions! If you would like to help improve the map:

1. **Make your edits** using the project files provided.
2. **Align with our sources:** Ensure your changes match the discussions on our Discord server or reference materials found in the `Reference_Images` folder (Discord discussions are preferred, but reference-based edits are also happily accepted!).
3. **Submit a Pull Request** with your updated version for review.
