## How it works:

* Files for QGIS are found in the QGIS_files folder. The **.qgz** file is the project file, useful for making changes to the map. The **.gpkg** file is the actual map file, useful for visualization or use with other mapping software.
* Editing of these files is permitted, and encouraged, and please submit a pull request if you believe your version is more accurate to the discussed plans on our discord (link below), or references found in the "Reference_Images" folder (not preferred, but still happily accepted)
